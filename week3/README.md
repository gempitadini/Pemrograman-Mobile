Program ini dibuat menggunakan bahasa pemrograman Dart untuk mengimplementasikan konsep dasar pemrograman serta penerapan version control menggunakan Git.

Program memiliki beberapa fitur utama, yaitu:
1. Menerima input angka dari pengguna melalui terminal.
2. Mengecek apakah angka yang dimasukkan termasuk bilangan prima atau bukan.
3. Menampilkan daftar seluruh bilangan prima dari angka 1 hingga angka yang dimasukkan.

Program disusun secara bertahap menggunakan sistem branching pada Git untuk menunjukkan proses pengembangan fitur secara terstruktur.
